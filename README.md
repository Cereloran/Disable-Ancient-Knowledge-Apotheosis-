# Disable-Ancient-Knowledge-Apotheosis-
A Minecraft Data pack for disabling a particularly annoying to remove "feature" of an otherwise amazing mod.

## Ancient Knowledge Removal

This datapack disables **Ancient Knowledge / Knowledge of the Ages** from Apothic Enchanting and Apotheosis.

### What it does
- Prevents `knowledge_of_the_ages` from being rolled normally
- Removes the Apotheosis melee affixes that apply the `apothic_attributes:knowledge` buff on hit
- Stops the XP-gain buff from being gained through normal enchantment and loot paths

### What it does not do
- It does not remove the mob effect from the mod itself
- It does not automatically strip the effect from items that already have it
- Existing gear may still need to be rerolled or replaced
